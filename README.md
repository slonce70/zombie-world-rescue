# 🧟 Операція: Порятунок Світу

3D-екшен проти зомбі прямо в браузері. Зомбі захопили планету — звільняй країни одну за одною! Перша країна: 🇺🇦 Україна.

## Як запустити

1. Відкрий термінал у цій папці та запусти сервер:
   ```
   python3 -m http.server 8741
   ```
2. Відкрий у браузері: **http://localhost:8741**
3. Натисни на жовту країну (Україна) на глобусі — і вперед!

> Гра працює повністю офлайн — нічого не завантажується з інтернету.

## Керування

| Клавіша | Дія |
|---|---|
| **WASD** | рух |
| **Мишка** | огляд і стрільба |
| **Shift** | біг |
| **Space** | стрибок |
| **E** | взаємодія (двері, ремонт, ящики) |
| **R** | перезарядка |
| **1 / 2** | вибір зброї |
| **V** | вид від 1-ї ↔ 3-ї особи |
| **B** | магазин |
| **M** | звук увімк/вимк |
| **Esc** | пауза |

## Що робити

1. 🆘 **Врятуй людей у хліві** — звільнені дають бонуси (медик лікує тебе поблизу!)
2. 📡 **Полагодь радіовежу** — тримай E, відбиваючись від зомбі
3. 📦 **Зачисть склад зброї** — нагорода: автомат!
4. Після кожного завдання — **орда** 🧟🧟🧟
5. Коли все виконано — відкривається арена з 👑 **Зомбі-Королем Буль-Булем**
6. Переможи його — і Україна звільнена! 🇺🇦

Монети за зомбі та завдання витрачай у магазині (B): аптечки, здоров'я, швидкість, шкода, патрони. Прогрес і прокачування зберігаються автоматично.

## Для розробника

- `test/smoke.mjs`, `test/e2e.mjs` — автотести (потрібен `npm install`, сервер на :8741)
- `?test&fresh&country=UKR` — URL-параметри для тестового режиму
- Все процедурне: моделі, світ, звук — жодного зовнішнього ассета, лише Three.js у `vendor/`
