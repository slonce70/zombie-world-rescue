# 🧟 Операція: Порятунок Світу

3D-екшен проти зомбі прямо в браузері. Зомбі захопили планету — звільняй країни одну за одною!

🎮 **Грати онлайн:** https://slonce70.github.io/zombie-world-rescue/

Працює на комп'ютері **та на телефоні/планшеті** (віртуальний джойстик і кнопки з'являються автоматично).

## Кампанія — у кожної країни своя унікальна карта!

| Країна | Карта | Особливість | Нагорода |
|---|---|---|---|
| 🇺🇦 Україна | село Сонячне: 🌻 соняшники, ставок із пірсом, вітряк, кури | перший рівень | 🔥 Автомат |
| 🇵🇱 Польща | містечко Зимове: ринкова площа з ратушею, ⛸ замерзле озеро (ковзько!), руїни замку, залізничне депо | ⛄ сніговики-зомбі, бос **Король Мороз** | 💥 Дробовик |

У кожній країні: 3 завдання, орди, фінальний бос. Здобута зброя лишається назавжди, рекорди часу зберігаються.

## Що ще є на картах

- 🏠 **Будинки, в які можна заходити**: лут усередині, але в деяких ховається зомбі-сюрприз 😱
- 🧨 **Вибухові бочки** з ланцюговими реакціями
- 🏆 **Золотий зомбі** (⭐ на мінікарті) — тікає від тебе, дожени і отримай джекпот
- 🦘 **Батути** закидають на дахи — там сховані скарби
- 🪂 **Аеродропи** з припасами кожні ~2 хвилини
- ⚽ Футбольний м'яч, 🐔 кури / 🐇 зайці, дим з димарів, птахи
- ⚙️ Налаштування якості (Авто/Гарна/Швидка) на глобусі

## Як запустити локально

```
python3 -m http.server 8741
```
і відкрий **http://localhost:8741**. Все офлайн — жодних завантажень.

## Керування (ПК)

| Клавіша | Дія |
|---|---|
| **WASD** | рух |
| **Мишка** | огляд і стрільба |
| **Shift** | біг |
| **Space** | стрибок |
| **E** | взаємодія |
| **R** | перезарядка |
| **1/2/3** | зброя (Q — наступна) |
| **G** | 💣 граната |
| **V** | вид від 1-ї ↔ 3-ї особи |
| **B** | магазин |
| **M** | звук |
| **Esc** | пауза |

На телефоні: лівий бік екрана — джойстик (до упору = біг), правий — огляд, кнопки праворуч — вогонь/стрибок/граната/перезарядка/взаємодія.

## Фішки

- 🔥 **Комбо**: серії вбивств дають бонусні монети
- 💣 **Гранати** з фізикою відскоку
- 🏆 **Рекорди** часу по кожній країні
- 🛒 Магазин (B): аптечки, прокачки здоров'я/швидкості/шкоди, патрони, гранати
- 💚 Врятований медик лікує тебе поблизу
- Все згенеровано кодом: моделі, світ, звук — жодного зовнішнього ассета

## Для розробника

- Тести: `node test/e2e.mjs` (Україна), `test/update2.mjs` (Польща + нові фічі), `test/flows.mjs`, `test/replay.mjs` (потрібні `npm install` і сервер на :8741)
- URL-параметри: `?test&fresh&country=POL&touch&seed=N`
- Дебаг-API: `window.__game.test.*`
- Нова країна = запис у `src/countries.js` (біом, складність, бос, нагорода) + lat/lon для глобуса
