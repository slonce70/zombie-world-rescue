# Штаб Рятівника — Roadmap (гібрид двох аудитів)

> **Для агентів-виконавців:** ОБОВ'ЯЗКОВА ПІД-СКІЛ: `superpowers:subagent-driven-development` (рекомендовано) або `superpowers:executing-plans` — виконувати по одній задачі за раз із рев'ю між ними. Кроки позначені чекбоксами (`- [ ]`).
>
> **Зв'язок з документами:** цей roadmap — НЕ заміна, а продовження дизайн-спеку [docs/superpowers/specs/2026-06-20-rescue-hq-design.md](../specs/2026-06-20-rescue-hq-design.md) і research-звіту [docs/research/2026-06-20-next-big-update.md](../../research/2026-06-20-next-big-update.md). Spec = стратегія («Штаб»), research = пріоритети («колекція дешева й сильна»), цей документ = **послідовність реалізації малими шматками з браузерною ШІ-перевіркою**.

**Мета:** перетворити розрізнені системи гри на одну ясну дитячу пригоду — НЕ одним великим «Штабом», а ланцюжком маленьких самостійних оновлень, кожне з яких Влад може добудувати за вечір-два, ОДРАЗУ побачити на екрані, і кожне перевіряється ШІ у браузері (десктоп + мобілка + скріни).

**Стратегія:** «Штаб Рятівника» — це **рамка** (одна кнопка на глобусі), що росте маленькими «кімнатами». Перша кімната — «Моя пригода» (альбом країн + мої цифри) — найдешевша й дає миттєвий «вау». Розумний рекомендаційний рушій, повний 4-вкладочний хаб, конструктор героя, сезони й UGC — НЕ зараз.

**Технології:** vanilla ES-модулі (БЕЗ бандлера), Three.js у `vendor/`, усі ассети генеруються кодом, PWA (`sw.js`), i18n uk/en/ru, headless-тести `node test/*.mjs`, браузерна ШІ-перевірка через MCP-браузер.

## Global Constraints (діють для КОЖНОЇ задачі — копіювати дослівно)

- **Без бандлера.** Тільки ES-модулі через import-map; новий файл = новий рядок у `index.html` (якщо треба) і **обов'язково** в масиві `SHELL` у `sw.js`, інакше зламається офлайн/PWA.
- **Ассети тільки кодом.** Жодних зовнішніх картинок/моделей. Іконки = емодзі/прапори/CSS. **Без 3D-рендеру в картки** в усьому roadmap (3D-силует = окремий рендер-пайплайн, заборонено в цих оновленнях).
- **i18n: ключ словника = український рядок.** Кожен видимий рядок через `t('…')`; після додавання — внести в `src/i18n/en.js` і `src/i18n/ru.js`. EN/RU не повинні бачити українського тексту.
- **Безпека дітей (непорушно):** без вільного текстового чату; без голосу; без приватних повідомлень; без friend-graph у перших версіях; без платних випадкових нагород; без агресивних streak-ів і сорому за пропуск; без FOMO-таймерів; без збору email/віку/реального імені/геолокації. Ніки — короткі, фільтровані (`worker/nick.mjs`).
- **Збереження прогресу дитини священне.** Нові поля сейва додавати в `_newSave()` ([src/main.js:323](../../../src/main.js)) І валідувати за формою в `_loadSave()` ([src/main.js:333](../../../src/main.js)) — зіпсований/чужий/старий сейв НЕ має кидати виняток (інакше вічний краш-екран).
- **Версія бампиться у ТРЬОХ місцях синхронно:** `version.json` (`v`), `APP_VERSION` у `src/main.js`, `CACHE` у `sw.js` (`zr-cache-vNN`). Гейт: `node test/version-check.mjs` (потрібен сервер :8741).
- **Кооп — хост-авторитарний.** Якщо оновлення НЕ чіпає мережу — підтвердити «нуль змін у коопі». Якщо чіпає — версія звіряється при вході в кімнату, протокол (`PROTO_VERSION`, зараз 3) бампиться при зміні формату пакетів, untrusted-дані від піра клампляться на хості.
- **Дисципліна скоупу під 10-річного автора.** Кожне оновлення ріжеться до найменшого самостійного «зрізу», який дає видиму перемогу. Ніколи не пакувати 4 системи в один реліз.
- **УВАГА: паралельні сесії ШІ в цій теці.** Перед редагуванням звіряти файли заново; чужі зміни (автоверсії, фікси) зберігати й зливати.

## 1. Рішення: що уместно, що відкласти, що не робити

Зведення двох аудитів (мій research-звіт + дизайн-спек «Штаб») після їх зближення. Обидва ШІ зійшлися; нижче — підсумкове рішення.

| Ідея | Вердикт | Куди |
|---|---|---|
| Колекція / «заповни альбом» як здорове ядро | ✅ Уместно (обидва ШІ — №1) | **M1** |
| «Мої цифри» (lifetime-статистика) | ✅ Уместно, дешево | **M1** |
| «Моя пригода» — країни з печатями | ✅ Уместно, тепла подача | **M1** |
| Бестіарій (картки на емодзі, БЕЗ 3D) | ✅ Уместно як простий зріз | **M1** |
| Престиж після 30 рівня | ✅ Уместно (мертвий XP — реальна діра) | **M1** |
| «Моя ціль» (мета над магазином) | ✅ Уместно — найсильніша оригінальна ідея спеку | **M2** |
| Глава 1 «Я рятівник» (м'який онбординг) | ✅ Уместно — закриває онбординг-діру | **M3** |
| «Поклич друга» (інвайт-посилання) | ✅ Уместно, інфра вже є (`?coopjoin=`) | **M4** |
| Безпечні пінги/емоції | ✅ Уместно, АЛЕ чіпає протокол | **M5** |
| Конструктор героя «Зроби свого» | ✅ Уместно, але це окрема велика гілка | **M6** |
| «Світ у вогні» (опційні зірки складності) | ⚠️ Тільки опційно, без тиску, не раніше | **M7** |
| Повний 4-вкладочний Штаб одразу | ❌ Не зараз (скоуп = «4 проєкти в одному») | Виростити з M1–M3 |
| «Сьогодні» з розумними картками (рекомендаційний рушій) | ❌ Не зараз («дорослий», мутний, нудно будувати) | Пізніше, після M3 |
| Сезонні події | ❌ Не зараз (обов'язок постійно оновлювати) | Після рамки |
| Звіринець як велика система петів | ❌ Не зараз (розпорошує фокус, багато нових моделей) | Песика — у M1 як частину альбому |
| UGC-редактор фортів / конструктор босів | ❌ Не зараз (модерація, безпека, scope L) | Далека мрія |
| Публічна галерея UGC | ❌ Ніколи без модерації | — |
| Лутбокси за гроші / FOMO / streak-кари / чат | ❌ Ніколи (обидва ШІ згодні) | — |

**Суть:** «що» зі спеку (колекція + смисл цілі + онбординг) — уместно й місцями краще; «скільки одразу» (весь Штаб як один апдейт) — не уместно для дитини-автора. Беремо ідеї, ріжемо скоуп.

### M1 — «Міні-Штаб: Моя пригода» 🎖️

## 4. Послідовність і чому саме так

```
M1 Моя пригода ─→ M2 Моя ціль ─→ M3 Глава 1 ─→ M4 Поклич друга ─→ M5 Пінги ─→ M6 Конструктор героя ─→ M7 Світ у вогні
   (соло, дешево)   (соло)        (соло)          (мережа, дешево)   (мережа⚠)   (велика гілка)          (обережно, опційно)
```

- **M1-M3 — соло.** Нічого не ламають у крихкому мережевому шарі, дають швидкі видимі перемоги, будують «рамку» поступово. Це паливо для ентузіазму дитини-автора.
- **M4 раніше за M5/M6** — найдешевша віральність, інфра вже є; її можна навіть підтягнути вперед, якщо хочеться росту раніше.
- **M5 — перша мережева правка** (протокол), тому після того, як соло-частина стабільна.
- **M6-M7 — більші/ризикованіші гілки**, кожна ріжеться на власні зрізи.
- **Не зараз:** повний Штаб одразу, «Сьогодні»-рушій, сезони, UGC-форти/боси, 3D-трофейна, публічна галерея, петоводство як окрема система, реферал-нагороди з соц-тиском.
